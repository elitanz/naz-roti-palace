# Naz Roti Palace — Miramar, FL

A handcrafted website for **Naz Roti Palace**, a family-run Trinidadian (Indo-Caribbean) roti
shop in Miramar, Florida — known for fresh-made roti, curry goat, and doubles. Built from the
business's real public data (Facebook, Yelp, and RestaurantJI listings and reviews).

🔗 **Live site:** https://elitanz.github.io/naz-roti-palace/

## About the business
- **Cuisine:** Trinidadian / Indo-Caribbean (roti, doubles, curry, dalpuri)
- **Address:** 6329 Miramar Pkwy, Miramar, FL 33023
- **Phone:** (954) 987-3964
- **Hours:** Mon–Fri 11:30 AM–8 PM · Sat 11:30 AM–7 PM · Sun closed
- **Rating:** 4.8★ from 158 reviews
- **Web presence before this:** Facebook + Instagram only — no website (RestaurantJI confirms "None listed")

## Design
A single, self-contained `index.html` (no build step). Distinct Indo-Caribbean identity:
cocoa brown + turmeric gold + scotch-bonnet orange + callaloo green, a signature **madras-check**
band (a nod to Trinidad's Indo-Caribbean heritage), warm Chonburi display type, and a Pacifico
script for the Trini call-outs (*Taste ah Trini!*, *Come lime with we!*). Fully responsive,
SEO-ready (Open Graph + JSON-LD Restaurant schema), scroll-reveal motion, working `tel:` /
directions / Facebook links, and an embedded map.

## Notes on data
All details and review content are drawn from the business's real public listings; menu items
reflect dishes documented there (curry goat roti, curry chicken roti, beef roti, doubles, aloo
pie, dalpuri, channa, pumpkin). Food photography is licensed stock (Pexels) standing in for the
restaurant's own photos — the natural first upgrade.
